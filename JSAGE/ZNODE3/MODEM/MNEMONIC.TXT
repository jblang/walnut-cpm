                 PC Pursuit Service Availability      
                 --------------------------------      
PC Pursuit can be used to access local numbers in the following cities.
Note that the access method is slightly different than standard in a
couple of cases.  Note that not all exchanges in a given area code are
accessible via PC Pusuit!
For a list of exchanges in the individual cities, please see the exchange
lists.      
Example of use: (old)     C DIAL202/12,<id>,<pw>         
               (*new*)    C D/DCWAS/12,<id>,<pw>
