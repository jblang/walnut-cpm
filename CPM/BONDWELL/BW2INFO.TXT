These tables give the basic port and pin information for the
Bondwell 2 laptop computer.  Expansion slot pins are not given
here because schematics are needed to make the information useful.
                                                      5/15/87

PORT PURPOSE                   PORT PURPOSE
---- -----------------------   ---- ----------------------
 00H parallel port A (8255)     30H slot select port       
 01H parallel port B (8255)     40H serial data reg (8251) 
 02H parallel port C (8255)     41H serial control reg     
 03H parallel control reg       50H printer output port    
 10H timer cntr reg 0 (8253)    60H floppy contrlr cmd reg 
 11H timer cntr reg 1 (8253)    61H floppy contrlr trk reg 
 12H timer cntr reg 2 (8253)    62H floppy contrlr sec reg 
 13H timer control reg          63H floppy contrlr data reg
 20H LCD controller data reg    70H internal modem tone sel
 21H LCD controller inst reg
