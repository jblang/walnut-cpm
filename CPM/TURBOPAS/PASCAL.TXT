     Delete
____________________________________________________________________________
Syntax: Delete( St, Pos, Num )

Deletå removeó á substrinç containinç 'Num§ characteró froí 'St§ startinç aô 
positioî 'Pos'®  'St§ ió á strinç variablå anä botè 'Pos§ anä 'Num§ arå 
integeò expressions®  Iæ 'Pos§ ió greateò theî 'Length¨ Sô )'¬ nï characteró 
arå removed®  Iæ aî attempô ió madå tï deletå characteró beyonä thå enä oæ 
thå strinç (i.e® 'Pos§ « 'Num§ exceedó thå lengtè oæ thå string)¬ onlù 
characteró withiî thå strinç arå deleted®  Iæ 'Pos§ ió outsidå thå rangå 
1..255 a run time error occurs.


     Insert
____________________________________________________________________________
Syntax: Insert( Obj, Target, Pos )

Inserô insertó thå strinç 'Obj§ intï thå strinç 'Target§ aô thå positioî 
'Pos'®  'Obj§ ió á strinç expression¬ 'Target§ ió á strinç variable¬ anä 
'Pos§ ió aî integeò expression®  Iæ 'Pos§ ió greateò thaî 'Length¨ Targeô

'Target§ wilì onlù contaiî thå left-mosô characters®  Iæ 'Pos§ ió outsidå

     Str

storeó thå resulô iî 'St'®  'Value§ ió á writå parameteò oæ typå integeò oò

     Val
____________________________________________________________________________

Vaì convertó thå strinç expressioî 'St§ tï aî integeò oò á reaì valuå 
(dependinç oî thå typå oæ thå variablå 'Var'© anä storeó thió valuå iî

Š     Seek
____________________________________________________________________________
Syntax: Seek( FilVar, n )

Seeë moveó thå filå pointeò tï thå n'tè componenô oæ thå filå denoteä bù 
'FilVar'® 'n§ ió aî integeò expression®  Thå positioî oæ thå firsô componenô 
ió zerï (0)®  Notå thaô iî ordeò tï expanä á filå iô ió possiblå tï seeë onå 
component beyond the last component.  The statement

     Seek( FilVar, FileSize( FilVar) );

will place the file pointer at the end of the file.



     Flush
____________________________________________________________________________
Syntax: Flush( FilVar )

Flusè emptieó thå internaì sectoò buffeò oæ thå disë filå 'FilVar'¬ anä thuó 
assureó thaô thå sectoò buffeò ió writteî tï thå disë iæ anù writå 
operationó havå occureä sincå thå lasô disë update®  Flusè alsï insureä thaô 
thå nexô reaä operatioî wilì actuallù perforí á physicaì reaä froí thå disë 
file.  Flush should never be used on a closed file.



     Close
____________________________________________________________________________
Syntax: Close( FilVar )

Thå disë filå associateä witè 'FilVar§ ió closed¬ anä thå disë directorù ió 
updated to reflect the new status of the file.



     Erase
____________________________________________________________________________
Syntax: Erase( FilVar )

Thå disë filå associateä witè 'FilVar§ ió erased®  Iæ thå filå ió open¬ iô 
should be closed before Erase is performed.



     Rename
____________________________________________________________________________
Syntax: Rename( FilVar, Str )

Thå disë filå associateä witè 'FilVar§ ió renameä tï á ne÷ namå giveî bù thå 
strinç expressioî 'Str'®  Thå disë directorù ió updateä tï sho÷ thå ne÷ namå 
oæ thå file¬ anä furtheò operationó oî 'FilVar§ wilì operatå oî thå filå 
with the new name.  Rename should not be used on an open file.

Noteº  Iô ió thå programmer'ó responsibilitù tï makå surå thå ne÷ filå namå 
doeó noô allreadù exisô oî thå disk¬ otherwiså multiplå occuranceó oæ thå 
file name will occur.
